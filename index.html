FashionMerge - Fashion & Music Fusion Hub

Overview: TrendSphere is a dynamic platform that combines fashion and music, allowing users to explore trending styles, create outfits, listen to personalized music playlists, and interact with a chatbot for recommendations.

Features

1. Fashion Module

  • Virtual Outfit Styler: AI-powered outfit suggestions based on season, event, and preference.
  • Trend Tracking: View trending outfits from influencers and brands.
  • Style Polls & Challenges: Vote on outfits, participate in daily fashion contests.
  • Personal Wardrobe Management: Save and organize favorite outfits.

2. Music Module

  • Mood-Based Playlist Generation: Personalized music recommendations.
  • Real-Time Listening Parties: Create or join live music sessions with friends.
  • Music Trivia & Fun Facts: Chatbot shares music-related insights.
  • Listening History & Analytics: Track most played genres and artists.

3. Chatbot Integration

  • Personalized Recommendations: Suggests outfits and playlists based on user behavior.
  • Interactive Conversations: Engages users with fashion and music trivia.
  • Challenge & Poll Participation: Helps users participate in style competitions.

4. User Profile & Analytics

  • Profile Customization: Personalized user profiles with badges.
  • Trend Analytics: View top-voted styles and playlists.
  • Wardrobe & Music History: Track past fashion and music preferences.

5. Backend & Technical Features

  • Authentication: Secure login using oauth by google.
  • Database: mongodb for structured data and AWS S3 or other(not decided yet) for media storage.
  • Real-Time Updates: WebSockets for live polls and music rooms.
  • Performance: Redis caching for fast trend retrieval.
